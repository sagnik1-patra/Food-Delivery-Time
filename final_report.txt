
Food Delivery Time Prediction using CNN and Model Validation

Author:
Sagnik Patra

Objective:
Predict whether a food delivery is Fast or Delayed using location, weather, traffic,
vehicle, rating, order priority, distance, and time-based features.

Dataset Loaded:
Food_Delivery_Time_Prediction.csv

Dataset Shape:
(200, 24)

Preprocessing Completed:
1. Dataset loaded successfully.
2. Missing numerical values handled using median imputation.
3. Missing categorical values handled using most frequent imputation.
4. Categorical features encoded using One-Hot Encoding.
5. Numerical features normalized using StandardScaler.

Feature Engineering Completed:
1. Customer coordinates extracted.
2. Restaurant coordinates extracted.
3. Haversine distance calculated.
4. Rush hour feature created.
5. Order hour feature created.
6. Weather impact feature created.

Image-Based CNN Representation:
Tabular features were converted into square image-like matrices so that Conv2D CNN
layers could process the data.

CNN Model:
Conv2D, BatchNormalization, MaxPooling2D, Flatten, Dense, Dropout, and Sigmoid layers.
