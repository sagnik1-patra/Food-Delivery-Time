
Food Delivery Time Prediction using CNN and Model Validation

Author:
Sagnik Patra

Objective:
The goal of this project is to predict whether a food delivery will be Fast or Delayed
using delivery, location, weather, traffic, rating, vehicle, priority, and time-based features.

Dataset:
Food_Delivery_Time_Prediction.csv

Note:
The dataset is used only for model training and is not included in the final submission package.

Dataset Shape:
(200, 23)

Median Delivery Time:
72.775

Target Creation:
Fast = Delivery_Time <= Median Delivery_Time
Delayed = Delivery_Time > Median Delivery_Time

Preprocessing:
1. Missing numerical values were handled using median imputation.
2. Missing categorical values were handled using most frequent imputation.
3. Categorical variables were encoded using One-Hot Encoding.
4. Numerical variables were normalized using StandardScaler.

Feature Engineering:
1. Customer and restaurant coordinates were extracted.
2. Haversine distance was calculated.
3. Rush hour feature was created.
4. Order hour was extracted.
